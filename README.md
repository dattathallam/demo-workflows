# demo-workflows

Exercises all four GitHub Actions types in a single workflow.

| Action | Type | Repo |
|--------|------|------|
| Node greeter | JavaScript / Node.js | [gh-action-node](https://github.com/dattathallam/gh-action-node) |
| File inspector | Docker container | [gh-action-docker](https://github.com/dattathallam/gh-action-docker) |
| Word counter | Go binary via Docker | [gh-action-go](https://github.com/dattathallam/gh-action-go) |
| Composite greeter | Composite | [gh-action-composite](https://github.com/dattathallam/gh-action-composite) |
